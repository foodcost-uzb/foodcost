# FOOD COST — Handoff документация

## Обзор проекта

**FOOD COST** — лендинг для компании, предоставляющей услуги управленческого учёта в сфере HoReCa (рестораны, кафе, общепит).

**URL:** http://localhost:3000

## Компоненты

### Logo.tsx
SVG-логотип с двумя вариантами:
- `variant="full"` — полный логотип с текстом "FOOD COST"
- `variant="icon"` — только иконка FC

Размеры: `sm`, `md`, `lg`

**Структура логотипа:**
- 3 концентрические рамки (тёмная → средняя → яркая фиолетовая)
- Буква F — тонкие линии (серый)
- Буква C — дуга с градиентом (розово-фиолетовый)
- 3 точки внизу (увеличиваются слева направо)

### Header.tsx
- Фиксированный header с blur-эффектом при скролле
- Навигация: Услуги, Продукты, О нас, Подкаст, Кейсы, Контакты
- Телефон + WhatsApp кнопка
- Мобильное меню (hamburger)

### Hero.tsx
Главный экран с заголовком, подзаголовком и CTA кнопками.

### Services.tsx
Карточки услуг с иконками и описаниями.

### Calculator.tsx
Интерактивный калькулятор для расчёта стоимости услуг.

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-сборки
npm start
```
